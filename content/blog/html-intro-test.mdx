# HTML简介

HTML（超文本标记语言）是构建网页的基础。它通过各种**标签**来描述网页的结构和内容。

## 常用HTML标签

- `<div>`：用于布局的块级元素。
- `<span>`：用于内联文本的标签。
- `<header>`、`<footer>`、`<section>`、`<article>`：语义化结构标签。

## HTML的语义化

语义化标签有助于提升网页的可访问性和SEO。
