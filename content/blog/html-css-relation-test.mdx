# HTML与CSS的关系

HTML用于定义网页的结构，而CSS用于控制网页的样式和布局。

## HTML标签与样式

- HTML标签如`<div>`、`<section>`用于结构化内容。
- CSS通过选择器（如`.class`、`#id`）为HTML元素添加样式。

## 实践示例

```html
<div class="container">Hello HTML & CSS</div>
```

## 结论

HTML和CSS共同构建了现代网页的基础。
